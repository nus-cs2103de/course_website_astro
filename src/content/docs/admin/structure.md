## Weekly Schedule

### Lectures
- **When**: Fridays 4:00 PM - 6:00 PM
- **Mode**: Hybrid (F2F and online)
- **Format**: Interactive discussions, live coding, Q&A

### Tutorials
- **When**: Various slots throughout the week
- **Duration**: 50 minutes
- **Format**: Small group discussions, peer learning

## Course Components

### 1. Individual Project (iP)
- Build a personal task manager (Duke)
- Learn Git, Java, Testing, Documentation
- Weeks 2-6

### 2. Team Project (tP)
- Enhance an existing product (AddressBook)
- Practice teamwork, project management
- Weeks 3-13

### 3. Weekly Topics
- Software Engineering principles
- Development tools and practices
- Testing and quality assurance
- Project management
