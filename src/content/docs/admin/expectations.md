## Learning Outcomes

Students will learn to:
- Build production-quality software
- Work effectively in teams
- Follow industry-standard practices
- Manage complex software projects

## Workload

This course requires approximately 10-12 hours per week including:
- Lectures (2 hours)
- Tutorials (1 hour)
- Project work (5-7 hours)
- Self-study (2-3 hours)

## Assessment

- Individual Project (iP): 30%
- Team Project (tP): 45%
- Participation: 10%
- Final Exam: 15%
